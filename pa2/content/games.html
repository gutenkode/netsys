Prompt - A hardcore platformer made with a lot of love.

Description
Prompt is a simple, clean, hardcore platformer. It was made in Java using LWJGL and OpenGL, and is powered by my own custom-built game engine Mote. It makes heavy use of shaders and postprocessing effects to give the impression of an old DOS-era command prompt. The gameplay is not for the faint of heart.

Rail Shooter - A retro-styled adventure in frustration.

Description

Scott Hirons vs. The World - A satirical game about the best teacher in my school.

Description
A satirical game about my junior year US History teacher. This was one of the first major projects I made with OpenGL. It's rough around the edges and has a lot of quirks, but this was the first time I really worked hard at finishing a game. It's something special to me, and I hope you can have fun with it too.
